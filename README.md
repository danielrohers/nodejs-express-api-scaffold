## Setup

**Requirements**

> [MongoDB](https://www.mongodb.org)

> [Node.js](https://nodejs.org)

**Install dependencies**
```bash
$ [sudo] npm install
```
or
```bash
$ yarn
```

## Launching

First start MongoDB if you don't have a running instance
```bash
$ mongod
```

**Development**
```bash
$ npm start
```
or use [nodemon](https://www.npmjs.com/package/nodemon)
```bash
$ npm run dev
```

**Production**
```bash
$ node bin/www
```
