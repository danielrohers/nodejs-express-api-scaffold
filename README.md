## Setup

### Installation

> [Install MongoDB](https://www.mongodb.org/downloads)

> [Install Node.js](https://nodejs.org/en/download)

```bash
[sudo] npm install
```


## Launching

First start MongoDB if you don't have a running instance

```bash
mongod
```

#### Development

```bash
npm start
```

or use [nodemon](https://www.npmjs.com/package/nodemon)

```bash
npm run nodemon
```

#### Production

```bash
node bin/www
```
